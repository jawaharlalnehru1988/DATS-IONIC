About Simple Rich Text Editor A lightweight editor focused on content pasting

This simple rich text editor is designed specifically for pasting content from other platforms like MS Word or Google Docs. It cleans up the HTML to include only basic formatting tags.

Key Features

  • Paste content from MS Word, Google Docs, etc.
  • Clean HTML output with only basic formatting tags
  • Preserves basic formatting (bold, italic, lists, etc.)
  • Removes complex styling and unwanted tags
  • Lightweight and easy to use
